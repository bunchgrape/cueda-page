2. Installation & Setup

ToolName is a web-based application, requiring no local installation. Simply navigate to the Run Tool page to begin.

Prerequisites

  • Modern web browser (Chrome, Firefox, Edge, Safari).
  • Data must be in CSV, TSV, or JSON format.

3. Data Preparation

For optimal results, ensure your input file adheres to the following structure. 

ID, FeatureA, FeatureB, Result\n1, 0.5, 12, True\n2, 1.2, 8, False

4. Running an Analysis

  1. Click the "Upload Data" button.
  2. Select your prepared file.
  3. Choose an analysis mode (Standard, Deep Dive, or Custom).
  4. Click "Process." Results will appear in the output panel below.

5. Interpreting Output

The output panel provides a summary of key metrics (p-values, R-squared) and a suite of interactive visualizations. Pay special attention to the *Confidence Score* which indicates data quality.

6. Advanced Features

Explore the **Model Tuning** section to adjust hyperparameters for niche applications. You can also export custom R-scripts used in the backend for full reproducibility.

7. Troubleshooting

If you encounter an `Error 404: Missing Header`, please verify that the first row of your CSV contains column names and not just data.
